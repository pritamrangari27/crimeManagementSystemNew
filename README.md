#  Crime Management System

##  Overview

The **Crime Management System** is a production-ready web application designed for law enforcement agencies, police stations, and the public. It enables citizens to file FIRs (First Information Reports) online, allows police officers to manage cases and criminals, and gives administrators full oversight of stations, users, and analytics вҖ” all through a secure, role-based platform.

>  **Live:** [https://crime-management-system-new.vercel.app/](https://crime-management-system-new.vercel.app/)

---

##  Features

###  For Citizens (Users)
-  **File FIRs online** with auto-generated FIR numbers (e.g. `FIR-2025-MH-MUM-00001`)
-  **AI-powered crime classification** вҖ” describe the incident, the system detects the crime type automatically
-  **Geo-tagged reports** with latitude/longitude support
-  **Real-time notifications** вҖ” get notified when your FIR is approved or rejected
-  **AI Chatbot** вҖ” 24/7 assistant for FIR guidance, safety tips, and legal FAQs
-  **Personal dashboard** вҖ” track all your filed FIRs and their statuses

###  For Police Officers
-  **Manage assigned FIRs** вҖ” view, approve, or reject cases
-  **Criminal database** вҖ” add, search, and update criminal records
-  **Station-based access** вҖ” officers only see data from their assigned station
-  **Case tracking** вҖ” full case history with workflow stages

###  For Administrators
-  **User management** вҖ” create, update, delete users with role assignment
-  **Bulk user import** вҖ” import up to 10,000 users via CSV
-  **Station management** вҖ” add and manage police stations
-  **Analytics dashboard** вҖ” crime heatmaps, monthly trends, FIR status charts
-  **Activity logs** вҖ” full audit trail of all system actions
